This demonstrates the new cancel button functionality for HTTP requests.

Status: Ready

How to Use

  • Click Send to start a request (default URL has 5 second delay)
  • The Cancel button appears while the request is in progress
  • Click Cancel to abort the request before it completes
  • The request immediately stops and UI resets

Test Scenarios

  • https://httpbin.org/delay/10 - 10 second delay (easy to cancel)
  • http://10.255.255.1 - Unresponsive host (will timeout, use cancel)
  • https://httpbin.org/drip?duration=20&numbytes=1000 - Slow streaming response

Visual States

  • Ready: Send button enabled, Cancel hidden
  • Loading: Send button disabled ("Sending..."), Cancel visible
  • Success: Both buttons reset to ready state
  • Cancelled: Shows "Request cancelled by user" message
